Code a simple webpage

Today, we're going to show you how to code a simple webpage using HTML.
Type the below code into Notepad (Windows) ,GEdit (Ubuntu)
<!DOCTYPE html>
<html>
<head>
<title> My First Webpage</title>
</head>
<body>
<h1>Hello :)</h1>
<p> I'm making my own HTML webpage with ForeverCoding! </p>
<p> You can make one too! forevercoding.weebly.com </p>
</body>
</html>

Next, save the file.

Name it as myfirstwebpage, select All Files, and then add the .html extension (see below.)
